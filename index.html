Definition de HTML

HyperText Markup Language HTML est le code utilisé pour structurer une page web et son contenu. Par exemple, le contenu de votre page pourra être structuré en un ensemble de paragraphes, une liste à puces ou avec des images et des tableaux de données. Comme le suggère le titre, cet article vous fournit les bases de compréhension du HTML et de ses fonctions.

Balises d en-tête

  • link: liaison avec une feuille de style
  • meta: métadonnées de la page web
  • script: code JavaScript
  • style: code CSS

Balises de structuration du texte

  1. abbr: abréviation
  2. blockquote: citation longue
  3. q: citation courte
  4. cite : citation du titre d une œuvre ou d un événement
  5. sub: mise en indice
  6. img: image grâce aux attributs src (adresse de l image) et alt (texte de remplacement)
  7. mark: mise en valeur visuelle (le texte est surligné)
  8. strong: mise en valeur forte (le texte est en gras)
